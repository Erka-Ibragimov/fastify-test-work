# Fastify Web Server

### ENV переменные

- `PG_USER` - пользователь базы данных PostgreSQL
- `PG_PASSWORD` - пароль базы данных
- `PG_HOST` - хост базы данных
- `PG_PORT` - порт базы данных
- `PG_DATABASE` - название базы данных

### Запуск

1. Установите зависимости:
   npm install
   npm run build
   npm run start
